# Git Bash

## 설정방법

설치 후 실행


### Git Bash 초기셋팅
``` git
git config

git config --global user.name "이름"

git config --global user.email "이메일주소"

git config --list
```

### Git Clone 
``` git
git clone "주소" .
```

### Git Add
``` git
git add . 
git commit -m "원하는 메세지"
git push
```
